1. Kenya voters visualization

This project aimed to test my data analysis and visualization skills. After downloading the data from the IEBC website which was in pdf form. I wrote a Python script to convert the data to a Comma Separated Value (CSV) file. Tapping the power of the pandas library I was able to clean the data while still maintaining the key columns values. Now data is clean for the next step which was visualization. My favorite visualization library is Plotly Express. I wrote a Python script to group the data into counties (u.s equivalents of states ) and I was to plot the distribution of voters in each constituency and location using the visualization of barplot,Sunburst Diagram , and pie chart. To ice the cake I used geopandas and geojson co-ordinates to plot the Choropleth map for voters.

2. British Airways Data Science Job Simulation

This project was a simulation of how data science skills are used in British Airways. This was an NLP project. The project aimed to do sentiment analysis for British airline reviews and present the findings to senior data scientist. 3.Boston Consulting Group Data science job simulation

PowerCo a major gas and electricity utility that supplies to corporate, SME, and residential customers has partnered with BCG to help diagnose the source of churning SME customers. The first task was writing an email to PowerCo to request data that I would need for the project and also in which format should the data be. After getting the data I did an exploratory data analysis to gain a holistic understanding of the dataset. Then I verified the hypothesis of price sensitivity being to some extent correlated with churn and sent my findings. After building the model I interpreted the model by deriving which features are the main contributors to customer churning.
